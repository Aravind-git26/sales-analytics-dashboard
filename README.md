#  Sales Analytics Dashboard

##  Project Overview
Interactive Sales Analytics Dashboard built using 
real Superstore dataset analyzing 9,994 transactions 
with business insights and KPI metrics.

##  Technologies Used
- Python
- Pandas
- Streamlit
- Plotly
- Data Analytics

##  Dashboard Features
| Feature | Description |
|---------|-------------|
| KPI Metrics | Sales, Profit, Orders, Margin |
| Category Analysis | Sales vs Profit by Category |
| Regional Analysis | Sales by Region & State |
| Sub-Category | Top 10 products |
| Profit Heatmap | Category vs Region |
| Discount Impact | Effect on Profit |
| Search Filter | City & State search |
| Raw Data Table | 9,994 transactions |

##  Key Insights
- Total Sales analyzed: $2.3M+
- Regions covered: 4 (East, West, North, South)
- Categories: Furniture, Office Supplies, Technology
- 17 Sub-Categories analyzed
